HTML5 video was meant to be this easy

html5media enables <video> and <audio> tags in all major browsers

All it takes is a single line of code to make HTML5 video and audio tags work in all major browsers.

  • Nothing to install.
  • Works in all browsers and phones.
  • One of the fastest solutions available.

Video & Audio tag demo

Depending on your web browser, the following video and audio files will be shown using either a HTML5 player, or a Flowplayer Flash fallback.

At the time of writing, the HTML5 player will be used in the Firefox, Safari, Chrome, Opera, Android and iPhone browsers.

The above video was embedded using a HTML5 <video> tag.

The above audio file was embedded using a HTML5 <audio> tag. Both tags can be enabled in all major browsers by including a single javascript file in your document.

Find out how to enable HTML5 video and audio tags in all major browsers at the html5media project site.
